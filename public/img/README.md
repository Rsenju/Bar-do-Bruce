# 📁 Imagens do Bar do Bruce

Esta pasta contém todas as imagens do site.
Coloque os arquivos **exatamente com estes nomes**:

## Hero
- `hero-bg.jpg` — Foto do ambiente/bar para o fundo do hero (recomendado: 1920×1080px mínimo)

## Sobre
- `sobre-ambiente.jpg` — Foto do interior do bar (recomendado: 800×1000px)
- `sobre-detalhe.jpg` — Foto de detalhe gastronômico (recomendado: 600×600px)

## Dicas
- Formato: `.jpg` ou `.webp`
- Tamanho recomendado para pratos: **800×533px** (proporção 3:2)
- O Next.js otimiza automaticamente via `next/image`
- Não usar `/public` no `src` — usar apenas `/img/nome.jpg`
